# Analisador Léxico

Projeto de Analisador Léxico desenvolvido para a disciplina de Compiladores da UFSJ - 2026/1.

Este projeto realiza a análise léxica de código-fonte em C, reconhecendo e classificando os principais tokens da linguagem.

O projeto inclui um makefile. Para compilar e executar, utilize os comandos: `make` seguido de `./main`.
